### **System Requirements**
- **Python**: 3.8 or higher
- **Operating System**: Windows, macOS, Linux
- **Memory**: 4GB RAM minimum, 8GB recommended

```bash

# Install dependencies
pip install -r requirements.txt

# Start web application
python web_app.py

# Start GUI application
python main.py
```
