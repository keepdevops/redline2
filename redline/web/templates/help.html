Help & Documentation

Getting Started
1. Download Data

Use the Download tab to fetch financial data from Yahoo Finance or Stooq.

2. View Data

Navigate to the Data tab to view and filter your downloaded datasets.

3. Convert Formats

Use the Converter tab to convert between different data formats.

4. Analyze Data

Perform analysis and generate insights in the Analysis tab.

Customization
Color Themes

Choose from multiple color-blind friendly themes using the Theme dropdown in the navigation bar.

Accessibility

All themes are designed to be accessible for users with various types of color blindness.

API Documentation
Data Operations
GET /api/files - List available files
GET /api/data/<filename> - Get data preview
POST /api/upload - Upload file
DELETE /api/files/<filename> - Delete file
System Operations
GET /api/status - System status
GET /api/health - Health check
GET /api/themes - Available themes
POST /api/theme - Set theme
Troubleshooting

  • Ensure file format is supported (CSV, JSON, Parquet, Feather, DuckDB)
  • Check file size limits
  • Verify file permissions
  • Try uploading smaller files first

  • Clear browser cache and reload
  • Check if localStorage is enabled
  • Try switching to a different theme
  • Verify browser supports CSS custom properties

  • Close unnecessary browser tabs
  • Check available memory
  • Try with smaller datasets
  • Clear browser cache
