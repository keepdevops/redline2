Help & Documentation

Getting Started
1. Download Data

Use the Download tab to fetch financial data from Yahoo Finance, Stooq, Alpha Vantage, or Finnhub.

2. View Data

Navigate to the Data tab to view, filter, and browse datasets with virtual scrolling for large files.

3. Convert Formats

Use the Converter tab to convert between CSV, Parquet, Feather, JSON, DuckDB, and TXT formats.

4. Analyze Data

Perform comprehensive analysis including financial, statistical, correlation, trend, volume, and price analysis.

Customization
Color Themes

Choose from multiple color-blind friendly themes using the Theme dropdown in the navigation bar.

Accessibility

All themes are designed to be accessible for users with various types of color blindness.

Getting Started
1. Register for an Account

Create your account to receive a license key and start using REDLINE.

  • • Visit the Registration page
  • • Enter your name, email, and company (optional)
  • • Receive your license key via email
  • • No installation required - access via web browser
2. Add Hours to Your Account

Purchase hours to use the platform. Pay only for what you use.

  • • Go to Settings or Payment tab
  • • Choose from available packages (5, 10, 20, 50 hours)
  • • Complete secure payment via Stripe
  • • Hours are added automatically
3. Access REDLINE

Access REDLINE from any device with a web browser.

  • • Open your web browser (Chrome, Firefox, Safari, Edge)
  • • Navigate to https://redfindat.com
  • • Enter your license key when prompted
  • • Start using REDLINE immediately
System Requirements
  • • Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
  • • Internet connection (broadband recommended)
  • • JavaScript enabled
  • • Cookies enabled for session management
API Documentation
Data Operations
GET /api/files - List available files
GET /api/data/<filename> - Get data preview
POST /api/upload - Upload file
POST /data/load-from-path - Load from file path
GET /data/browse - Browse file system
POST /data/stooq/download - Download Stooq data
Analysis & System
POST /analysis/analyze - Run analysis
POST /converter/convert - Convert format
GET /api/status - System status
GET /api/themes - Available themes
POST /api/theme - Set theme
GET /api/health - Health check
Troubleshooting

  • Ensure file format is supported (CSV, JSON, Parquet, Feather, DuckDB)
  • Check file size limits
  • Verify file permissions
  • Try uploading smaller files first

  • Clear browser cache and reload
  • Check if localStorage is enabled
  • Try switching to a different theme
  • Verify browser supports CSS custom properties

  • Close unnecessary browser tabs
  • Check available memory
  • Try with smaller datasets
  • Clear browser cache
